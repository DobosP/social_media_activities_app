How these become production (open-source, fast)

Type
Inter variable font (SIL OFL), self-hosted + system fallback — instant first paint, no Google-Fonts call.
Icons
Lucide (ISC) as an inline SVG sprite — no icon-font, only what's used.
Illustrations
unDraw (free, recolour to brand) or generated inline SVG like Grove's hero — tiny & crisp.
Interactivity
htmx + a little Alpine.js for fast server-rendered partials — no SPA, keeps your Django templates.
Theming
CSS custom properties; dark mode via a data-theme toggle (remembers via localStorage + honours prefers-color-scheme).
Delivery
One small CSS file served & compressed by WhiteNoise. Respects prefers-reduced-motion. Zero trackers.
