{% trans "How it updates" %}

{% trans "Venues and events are refreshed periodically from their sources (OpenStreetMap, Overture Maps, RO-EDU, venue calendars, and staff/community edits). There is no live push feed — poll the feeds or JSON API on a reasonable schedule (see the rate limit below)." %}

{% trans "Licensing & attribution" %}

{% trans "Venue and event records carry their own source attribution and licence where known — for example, OpenStreetMap contributors under the Open Database Licence (ODbL), the Overture Maps Foundation, or RO-EDU. Each place or event page, and the corresponding JSON API record, shows that record's specific attribution: check the record, not this page, for the authoritative credit. Where no source licence applies, content is offered on the same open terms as the rest of this nonprofit's public pages — free to read, cite and reuse with attribution." %}
